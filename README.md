## 🚀 Getting Started

### Prerequisites

- Node.js = v22.14.0
-  npm

### Environment Variables (.env)

```bash
NEXT_PUBLIC_SERVER_API_URL=
NEXT_PUBLIC_SERVER_BASE_URL_Backend_Backend=
NEXTAUTH_SECRET=
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
NEXTAUTH_URL=
NEXT_PUBLIC_SERVER_BASE_URL_Backend=
EMAIL_PASS=
EMAIL_USER=
EMAIL_NAME=
NEXTAUTH_URL=
GITHUB_ID=
GITHUB_SECRET=
STORE_ID=
STORE_PASSWORD=
IS_LIVE=
NEXT_PUBLIC_BASE_URL=
```

### Installation

```bash
git clone https://github.com/SHARIFULALAM2025/Travel_Planning.git
cd my-app
npm install
```

### Development

```bash
npm run dev
```

### Production Build

```bash
npm install -g vercel
vercel login
vercel --prod


```
